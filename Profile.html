Project Description:

Dự án web của tôi sử dụng HTML, CSS và jQuery cắt ghép giao diện từ photoshop .psd file để tạo ra một trang web động và thân thiện với người dùng. Với HTML, tôi xây dựng cấu trúc nội dung cơ bản của trang web và định dạng các phần tử như tiêu đề, đoạn văn và danh sách. CSS giúp tôi tạo ra các hiệu ứng đẹp mắt và bố cục trang theo ý muốn. Cuối cùng, tôi sử dụng jQuery để thêm các chức năng tương tác và hoạt ảnh vào trang web một cách dễ dàng. Kết quả cuối cùng là một trang web đẹp mắt, dễ sử dụng và thân thiện với người dùng.

Technology:

HTML5 , CSS 3 ,jQuery , Font Awesome
